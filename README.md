# Student-Portfolio-with-Daily-Blog-WSTlab2
A collaborative project built using HTML and CSS and managed through Git and GitHub. This website features a students personal information and daily activities.

The website will feature the students profile including their name, age, address, contact information, and skills and their daily blog showcasing their day-to-day activities until the submission of this project.
